## Cocoapods ##

### Installing Cocoapods ###

1. Open Terminal
2. Enter command: ```sudo gem install cocoapods```
3. Enter your password
4. Wait for this to finish, don't touch your terminal. It might take a couple minutes
5. Run command ```pod install```
